<h2> Solutions </h2>

**1. Z-scored Normalised data**

```python

df['zscore_age'] = (df.Age - df.Age.mean())/df.Age.std(ddof=0)
df['zscore_%fat'] = (df.perFat - df.perFat.mean())/df.perFat.std(ddof=0)

```
